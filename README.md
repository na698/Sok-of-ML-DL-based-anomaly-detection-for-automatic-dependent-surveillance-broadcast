# ADS-B Anomaly Detection using ML & DL

## 📌 Project Overview
This project focuses on detecting anomalies in Automatic Dependent Surveillance-Broadcast (ADS-B) aviation data using Machine Learning and Deep Learning techniques.

It helps identify unusual aircraft behavior, improving aviation safety and security.

---

## 🚀 Features
- Data preprocessing of ADS-B datasets
- Machine Learning model (Isolation Forest)
- Real-time anomaly detection dashboard (Streamlit)
- Visualization of aircraft anomalies
- Clean and labeled dataset handling

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- SQLite

---

## 📂 Project Structure
- `dashboard/` → Streamlit dashboard
- `data/` → Dataset files
- `src/` → ML models & preprocessing
